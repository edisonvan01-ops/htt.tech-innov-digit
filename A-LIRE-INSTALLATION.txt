TECH-INNOV DIGIT SERVICE - Site web Next.js

Fichiers à copier-coller :
- src
- public
- package.json
- pnpm-lock.yaml
- pnpm-workspace.yaml
- next.config.ts
- tailwind.config.ts
- postcss.config.mjs
- tsconfig.json
- next-env.d.ts
- .eslintrc.json
- .gitignore

Installation :
1. Ouvrir le dossier dans un terminal.
2. Installer les dépendances : pnpm install
3. Lancer le site en local : pnpm dev
4. Préparer la production : pnpm build
5. Lancer la version production : pnpm start
